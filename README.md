# AWS Multi-Tier VPC Infrastructure Project

## Project Overview

Designed and implemented a production-style AWS infrastructure using custom VPC networking and multiple EC2 instances.

## Architecture Components

* Custom VPC (10.0.0.0/16)
* 2 Public Subnets
* 2 Private Subnets
* Internet Gateway
* Route Tables
* Security Groups
* Multiple Amazon EC2 Instances
* Automated Web Server Deployment using User Data

## Technologies Used

* AWS VPC
* Amazon EC2
* Linux
* Bash Scripting
* HTTP
* SSH

## Project Objectives

* Understand AWS networking concepts.
* Build infrastructure manually without using the VPC Wizard.
* Configure secure network communication.
* Automate web server provisioning.

## Learning Outcomes

* Cloud Networking
* Infrastructure Deployment
* Security Configuration
* Linux Administration
* Troubleshooting
